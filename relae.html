SaaS Product UX Design

Relae Git is a product within Relae, a Productivity SaaS that provides Time Tracking and Synology File Management with other communication tool for Project Management.
As every company becomes a tech company, Relae Git explores the possibility of having Issue Tracking and Deployment Pipeline integrated within Relae.

The Product:
• Issue Tracking
• Deployment Pipeline
• Synology File Integration
